# Weather App

A responsive web application for checking weather conditions.

## Live Preview

Check out the live app here: https://nanoweather.netlify.app/

## Project Composition

This project is built with:
- **JavaScript** - 45.8%
- **CSS** - 38.4%
- **HTML** - 15.8%

## Features

- Real-time weather information
- Clean and intuitive user interface
- Responsive design

## Getting Started

To use this weather app locally:

1. Clone the repository
2. Open `index.html` in your web browser
3. Start checking the weather!

## Deployment

The app is live at https://nanoweather.netlify.app/
